# JCB請求メール抽出ツール セットアップ手順

## 必要なもの
- Python 3.9 以上
- Google アカウント（Gmailと同じもの）

---

## Step 1: Google Cloud Console でAPIを有効化

1. https://console.cloud.google.com/ を開く
2. 新しいプロジェクトを作成（例: `jcb-billing-extractor`）
3. 左メニュー「APIとサービス」→「ライブラリ」を開く
4. 以下の2つを検索して「有効にする」:
   - **Gmail API**
   - **Google Drive API**

---

## Step 2: OAuth 認証情報を作成

1. 「APIとサービス」→「認証情報」を開く
2. 「認証情報を作成」→「OAuthクライアントID」を選択
3. アプリの種類: **デスクトップアプリ** を選択
4. 名前を入力して「作成」
5. 「JSONをダウンロード」をクリック
6. ダウンロードしたファイルを **このフォルダ内に `credentials.json` という名前で保存**

---

## Step 3: OAuth同意画面の設定

1. 「APIとサービス」→「OAuth同意画面」を開く
2. ユーザーの種類: **外部** を選択して作成
3. アプリ名・メールアドレスを入力して保存
4. 「テストユーザー」にご自身のGmailアドレスを追加

---

## Step 4: 依存ライブラリをインストール

```bash
pip install -r requirements.txt
```

---

## Step 5: スクリプトを実行

```bash
python extract_jcb_billing.py
```

初回実行時にブラウザが開きます。Googleアカウントでログインして「許可」をクリックしてください。

---

## 出力ファイル

| ファイル | 内容 |
|---|---|
| `JCB請求履歴.xlsx` | 抽出結果のExcel（ローカル） |
| `token.pickle` | 認証トークン（自動生成・削除不要） |

Googleドライブにも同名ファイルが自動保存されます。2回目以降の実行では上書き更新されます。

---

## Excel の列構成

| 列 | 内容 |
|---|---|
| 受信日時 | メールを受け取った日時 |
| 件名 | メールの件名 |
| 送信元 | mail@qa.jcb.co.jp |
| 請求月 | 請求対象の月（例: 2025年04月） |
| 支払日 | 引落日（例: 2025/05/10） |
| 請求金額（円） | 抽出した請求金額 |

---

## 注意事項

- `credentials.json` と `token.pickle` は外部に公開しないでください
- 請求金額が「抽出不可」のメールはメール本文のフォーマットが想定外の可能性があります
  → その場合はご連絡ください。正規表現パターンを追加対応します
