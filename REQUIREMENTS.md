# 漢字チャレンジ 要件定義書

**バージョン:** 1.0
**作成日:** 2026年4月29日
**プロジェクト名:** 漢字チャレンジ（小学2年生向け漢字学習アプリ）
**リポジトリ:** shigeru222/shigeru

---

## 1. プロジェクト概要

### 1.1 背景・目的

小学2年生で習う配当漢字160字を、ゲーム感覚で楽しく繰り返し練習できるWebアプリを提供する。家庭学習や学童・通学前後の隙間時間に、保護者の付き添いがなくても子ども一人で取り組めることを目指す。

### 1.2 ターゲットユーザー

- 小学2年生（メインユーザー、6〜8歳）
- 漢字の先取り学習をしたい小学1年生
- 復習をしたい小学3年生以上
- 学習の様子を見守りたい保護者

### 1.3 解決する課題

| 課題 | 本アプリの解決策 |
|------|------------------|
| ドリルは飽きやすく続かない | カラフルなUI＋効果音風の絵文字フィードバックで楽しく続けられる |
| 答え合わせを保護者がしないといけない | 4択クイズは自動判定、書きクイズは自己評価で完結する |
| どの漢字を覚えたか把握しづらい | 連続正解で「⭐マスター」判定、進捗を可視化 |
| 紙のドリルを持ち歩けない | ブラウザだけで完結、iPadでもスマホでも使える |
| アプリのインストールが面倒 | URLを開くだけで起動 |

### 1.4 学習対象

- 文部科学省「学習指導要領」小学2年生配当漢字 **160字**
  （引・羽・雲・園・遠・…・里・理・話）
- 各漢字について、音読み、訓読み、やさしい意味、例語2つ、画数を収録

## 2. 機能要件

### 2.1 学習モード一覧

| モード | URL | 概要 |
|--------|-----|------|
| ホーム | `/` | 進捗サマリーと各モードへの入口 |
| よみクイズ | `/quiz?type=yomi` | 漢字（送り仮名付き語）の読みを4択で答える |
| いみクイズ | `/quiz?type=imi` | 漢字の意味を4択で答える |
| かきクイズ | `/write` | 読みを見て漢字をペンで書く（自己採点） |
| おくりがなクイズ | `/okurigana` | 漢字に続く送り仮名をひらがなで入力する |
| フラッシュカード | `/flashcard` | カードをめくって読み・意味を確認 |
| かんじいちらん | `/list` | 160字を一覧表示・絞り込み・詳細表示 |
| がくしゅうきろく | `/progress` | 進捗統計・マスター済み漢字の閲覧 |

### 2.2 ホーム画面

- アプリのタイトル「かんじチャレンジ！」を表示
- 学習進捗を表示
  - マスターした漢字の総数 / 160
  - 進捗バー（0〜100%）
  - 内訳：マスター数・練習中数・未着手数
- 6つのモードカード（タップで各モードへ遷移）

### 2.3 よみクイズ機能

#### 2.3.1 出題

- 訓読みを持つ漢字からランダムに10問選出
- 出題形式は「漢字＋送り仮名」を優先
  - 例：古い、走る、引く、強い、明るい
  - 該当する例語が無い漢字は単漢字＋訓読みで出題（雲→くも、馬→うま 等）
- 訓読みを持たない漢字（科・電・線 等）は対象外

#### 2.3.2 選択肢

- 正解を含む4択（ひらがな）
- 誤答は他の漢字の代表訓読みから抽出（重複なし）

#### 2.3.3 採点

- 自動採点
- 正解時：✅ 緑色＋「せいかい！」フィードバック
- 不正解時：❌ ピンク色＋正解の漢字・読み・意味・例を表示

### 2.4 いみクイズ機能

#### 2.4.1 出題

- 全160字からランダムに10問選出
- 単漢字を表示

#### 2.4.2 選択肢

- 正解を含む4択（やさしい日本語の意味）
- 誤答は他の漢字の意味からランダム抽出

#### 2.4.3 採点

- よみクイズと同じ自動採点・フィードバック

### 2.5 かきクイズ機能

#### 2.5.1 出題

- 訓読みを持つ漢字からランダムに10問選出
- 「『ふるい』を かんじで かこう」のように、ひらがなで読みを提示

#### 2.5.2 描画キャンバス

- 320×320px の正方形キャンバス
- 入力デバイス：Apple Pencil・指・マウスすべて対応（Pointer Events API）
- バランス練習用の十字ガイド線を常時表示
- 太字（10px）・線端・線結合は丸め

#### 2.5.3 学習補助機能

| 機能 | 動作 |
|------|------|
| 🔄 クリア | 書いた線をすべて消す |
| 💡 ヒント | 漢字の意味をひらがなで表示 |
| ✏️ なぞる | 正解漢字を10%透過で背景に表示し、なぞって練習できる |

#### 2.5.4 採点

- AIによる自動文字認識は行わない
- 「こたえを みる」ボタンで正解の漢字を表示
- 自分の書いた字と見比べて「かけた」「だめだった」を選択（自己評価）
- 自己評価結果は進捗に反映される

### 2.6 送り仮名クイズ機能

#### 2.6.1 出題

- 漢字＋送り仮名の例語を持つ漢字からランダムに10問選出
- 例：古い、走る、引く、強い、明るい、食べる、新しい、考える 等
- 単漢字のみ／訓読み無しの漢字（雲、馬、科 等）は対象外

#### 2.6.2 解答方法

- 「古□」のように漢字の右に空欄を表示
- ひらがなで送り仮名を入力（テキスト入力）
- 読みと、必要に応じて意味（ヒントボタン）を表示
- スペースは無視して完全一致で判定

#### 2.6.3 採点・フィードバック

- 自動採点（完全一致のみ正解）
- 不正解時：正解の語・読み・送り仮名・意味を表示
- 連続3回正解で「マスター」判定（他モードと共通）

### 2.7 フラッシュカード機能

#### 2.6.1 デッキ構成

- 全160字からランダムに20枚のカード

#### 2.6.2 カード操作

- 表面：漢字を大きく1文字表示
- タップで裏面に反転：意味・音読み・訓読み・代表例・画数
- 「おぼえてた！」「わからなかった」を選択して次のカードへ

### 2.8 かんじいちらく機能

#### 2.7.1 一覧表示

- 160字を4〜8列のグリッドで表示（画面幅に応じて調整）
- マスター済みの漢字には⭐マークと黄色背景

#### 2.7.2 絞り込み

| フィルタ | 対象 |
|----------|------|
| ぜんぶ | 160字すべて |
| ⭐マスター | 連続3回正解した漢字 |
| れんしゅう中 | 出題されたが未マスター |
| まだ | 一度も出題されていない漢字 |

#### 2.7.3 詳細モーダル

- 漢字をタップすると詳細を表示
- 表示項目：漢字、画数、意味、音読み、訓読み、例語2つ（ふりがな付き）

### 2.9 学習記録機能

#### 2.8.1 進捗の表示

- マスターした漢字数 / 160（％つき）
- 全体進捗バー
- 内訳カード（マスター・練習中・まだ）
- 統計：解いた問題数、正解率
- マスター済み漢字を一覧表示
- 進捗に応じた応援メッセージ

#### 2.8.2 リセット機能

- 「きろくを けす」ボタンで全進捗を初期化
- 確認ダイアログを表示

### 2.10 進捗管理ロジック

| 項目 | 仕様 |
|------|------|
| 出題回数 | クイズ・カード・書きクイズで各1問解くごとに+1 |
| 正解数 | 4択正解／カード「おぼえてた」／書き「かけた」で+1 |
| 連続正解数 | 連続で正解した回数（不正解で0にリセット） |
| マスター判定 | 連続3回正解で「マスター」（一度マスターすれば連続が途切れても維持） |
| 最終学習時刻 | 直近の出題タイムスタンプ |

## 3. 非機能要件

### 3.1 可用性

- インターネット接続後はオフラインでも動作（静的サイト）
- PC・スマートフォン・タブレットのブラウザから利用可能
- 特別なアプリインストール不要
- iPad＋Apple Pencilでの書き取り体験を主要シナリオの1つとして想定

### 3.2 パフォーマンス

- 初回ロード後、画面遷移は即時（クライアントナビゲーション）
- すべての画面が静的生成（プリレンダリング）
- 外部APIコールなし

### 3.3 セキュリティ・プライバシー

- 外部サーバーへのデータ送信なし
- 学習進捗はブラウザのlocalStorageのみに保存
- 個人情報（氏名・メール等）の入力・保存なし
- 認証機能なし

### 3.4 対応環境

| 区分 | 対応 |
|------|------|
| ブラウザ | Chrome・Safari・Firefox・Edge（最新版） |
| デバイス | PC・スマートフォン・タブレット |
| 入力 | キーボード・マウス・タッチ・ペン（Apple Pencil等） |

### 3.5 アクセシビリティ・子ども向け配慮

- 子ども向けのまるみのある書体（Hiragino Maru Gothic ProN）
- 大きめの文字とタップしやすい大型ボタン
- 主要なテキストはひらがな中心
- 明るく親しみやすい色（黄・桃・水色・緑・紫・橙）
- 太い縁取りと立体感のある影でクリックしやすさを表現
- 絵文字による視覚的フィードバック

## 4. システム構成

### 4.1 技術スタック

| 分類 | 技術 |
|------|------|
| フレームワーク | Next.js 16（App Router・Turbopack） |
| 言語 | TypeScript（strict mode） |
| スタイリング | Tailwind CSS v3 + カスタムCSS |
| 描画 | HTML5 Canvas + Pointer Events API |
| ホスティング | Vercel（Hobbyプラン想定） |
| データ保存 | ブラウザ localStorage |
| 外部API | なし |

### 4.2 ディレクトリ構成

```
app/
├── layout.tsx           ルートレイアウト（メタ情報、背景）
├── page.tsx             ホーム画面
├── globals.css          全体テーマ（カラー・ボタン・アニメーション）
├── quiz/page.tsx        よみ・いみクイズ
├── write/
│   ├── page.tsx         かきクイズ
│   └── DrawingCanvas.tsx 描画キャンバスコンポーネント
├── flashcard/page.tsx   フラッシュカード
├── list/page.tsx        漢字一覧
└── progress/page.tsx    学習記録

lib/
├── kanji-data.ts        160字の漢字データ
├── kanji-types.ts       型定義（Kanji, KanjiProgress, QuizQuestion）
├── kanji-storage.ts     localStorage 進捗保存
└── kanji-quiz.ts        共通ユーティリティ（getYomiPrompt, shuffle）
```

### 4.3 漢字データ構造

```typescript
type Kanji = {
  char: string;                 // 漢字
  on: string[];                 // 音読み（カタカナ）
  kun: string[];                // 訓読み（ひらがな）
  meaning: string;              // やさしい意味
  examples: { word: string; reading: string }[]; // 例語（読み付き）
  strokes: number;              // 画数
};
```

### 4.4 進捗データ構造

```typescript
type KanjiProgress = {
  seen: number;       // 出題された回数
  correct: number;    // 正解した回数
  streak: number;     // 連続正解数
  mastered: boolean;  // マスター済み（連続3回正解）
  lastSeen: number;   // 最後に学習した時刻（unix ms）
};

type AllProgress = Record<string /* 漢字 */, KanjiProgress>;
```

### 4.5 localStorage キー

| キー | 内容 |
|------|------|
| `kanji-progress-v1` | 全漢字の学習進捗（JSON文字列） |

## 5. 運用要件

### 5.1 デプロイ

- GitHub の `main` ブランチへの push で Vercel が自動デプロイ
- 環境変数の設定不要
- ビルドコマンド：`npm run build`
- 実行コマンド：`npm start`（本番）／`npm run dev`（開発）

### 5.2 ランニングコスト

- AI APIコールなし → 利用料金は発生しない
- Vercel Hobbyプラン（無料枠）で運用可能

### 5.3 バックアップ・データ移行

- 学習進捗はブラウザlocalStorageのみに保存
- ブラウザ／端末を変更すると進捗は引き継がれない
- データ消去操作（学習記録ページの「きろくをけす」）でリセット可能

## 7. 用語

| 用語 | 説明 |
|------|------|
| マスター | 連続3回正解した漢字。⭐マークが付与される |
| 練習中 | 1回以上出題されたが、まだマスターしていない漢字 |
| まだ | 一度も出題されていない漢字 |
| 配当漢字 | 学習指導要領で各学年に配分された漢字 |
| Pointer Events | マウス・タッチ・ペンを統一的に扱うブラウザAPI |
