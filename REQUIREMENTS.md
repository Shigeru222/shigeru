**バージョン:** 1.0  
**作成日:** 2026年4月26日  
**プロジェクト名:** 英検2級 AI模擬試験アプリ  
**URL:** https://shigeru-wq33.vercel.app

---

## 1. プロジェクト概要

### 1.1 背景・目的

英検2級の合格を目指す学習者が、自宅でいつでも本番形式の模擬試験・面接練習を行えるWebアプリを提供する。AIが問題を自動生成するため、毎回異なる問題で繰り返し練習できる。

### 1.2 ターゲットユーザー

- 英検2級の受験を控えた中学生・高校生
- 英検2級対策を行いたい社会人
- 保護者（子どもの学習をサポートしたい方）

### 1.3 解決する課題

| 課題 | 本アプリの解決策 |
|------|----------------|
| 市販の問題集は同じ問題を繰り返してしまう | AIが毎回異なる問題を生成 |
| 採点・解説に時間がかかる | 即座に自動採点・解説を提供 |
| 弱点がわかっても何をすべきかわからない | AIが弱点を分析し改善アクションを提案 |
| 面接練習の相手がいない | AIが面接官として練習できる |
| 環境構築が難しい | URLを開くだけで使える |

## 2. 機能要件

### 2.1 模擬試験機能

#### 2.1.1 試験問題の自動生成

- AIが英検2級レベルの試験問題を毎回異なる内容で生成する
- 生成される問題の構成は以下の通り

| セクション | 問題数 | 配点 | 内容 |
|-----------|--------|------|------|
| 語彙・文法 | 20問 | 20点 | 穴埋め形式・4択 |
| 読解 | 6問 | 6点 | 英文パッセージ＋設問・4択 |
| 英作文 | 1問 | 16点 | 80〜100語のエッセイ |
| **合計** | **27問** | **42点** | |

- 語彙問題の難易度：英検2級上位レベルの単語（exacerbate、alleviate等）
- 読解パッセージ：400〜500語、学術的・社会的トピック
- 英作文：社会的テーマについて意見＋2つの理由を記述

#### 2.1.2 採点機能

- 語彙・文法、読解：選択肢と正解を照合し自動採点
- 英作文：AIが以下の4観点で採点（各4点、計16点）
  - 内容（Content）
  - 構成（Organization）
  - 語彙（Vocabulary）
  - 文法（Grammar）
- 合格目安：42点満点中65%以上（約27点以上）

#### 2.1.3 解答解説機能

- 全問題について正解・不正解を表示
- 各問題に解説文を表示
- 英作文については観点別フィードバックを表示

#### 2.1.4 弱点分析・改善提案機能

- 試験結果をAIが分析し以下を提供
  - 全体評価コメント
  - 弱点エリア（カテゴリ・重要度付き）
  - 改善アクション（優先度・所要時間付き）
  - 2週間学習プラン
  - 励ましのメッセージ

### 2.2 面接練習機能

#### 2.2.1 面接問題の自動生成

- AIが英検2級二次試験（面接）形式の問題を生成
- 生成される問題の構成

| ステップ | 内容 |
|---------|------|
| ① 音読 | 3〜4文の英文パッセージを音読 |
| ② パッセージQ | パッセージの内容に関する質問（1問） |
| ③ イラスト描写 | 状況説明のイラストを英語で描写（1問） |
| ④⑤ 意見表明 | 社会的テーマについて意見＋理由（2問） |

#### 2.2.2 回答入力

- テキスト形式で英語回答を入力
- 模範回答をオプションで表示可能

#### 2.2.3 面接採点・フィードバック機能

- AIが各質問を4点満点で採点（合計16点）
- 採点基準：正確性・流暢さ・内容の充実度・文法
- 質問ごとに以下を提供
  - スコア
  - 良かった点
  - 改善点
  - より良い回答例
- 全体評価として強み・弱点・学習アドバイスを提供

### 2.3 学習履歴管理機能

- 模擬試験の結果を最大10件保存
- ホーム画面で直近3件の結果を表示
- 過去の結果を再閲覧可能

### 2.4 ナビゲーション機能

- ホーム画面から模擬試験・面接練習へ移動
- 試験・面接結果画面からホームへ戻るボタン
- 試験・面接終了後に再挑戦ボタン

## 3. 非機能要件

### 3.1 可用性

- インターネット接続があれば、PC・スマートフォン・タブレットのブラウザから利用可能
- 特別なアプリインストール不要

### 3.2 パフォーマンス

- 試験問題の生成：60秒以内
- 面接問題の生成：60秒以内
- 採点・分析：60秒以内

### 3.3 セキュリティ

- APIキーはサーバーサイドのみで管理（フロントエンドに露出しない）
- ユーザーの回答データはブラウザのlocalStorageのみに保存（外部サーバーへの送信なし）

### 3.4 対応環境

- 対応ブラウザ：Chrome、Safari、Firefox、Edge（最新版）
- 対応デバイス：PC、スマートフォン、タブレット

## 4. システム構成

### 4.1 技術スタック

| 分類 | 技術 |
|------|------|
| フレームワーク | Next.js 16（App Router） |
| 言語 | TypeScript |
| スタイリング | Tailwind CSS v3 |
| AIエンジン | Claude Opus 4.7（Anthropic） |
| AIクライアント | @anthropic-ai/sdk |
| ホスティング | Vercel（Hobbyプラン） |
| データ保存 | ブラウザ localStorage |

### 4.2 画面構成

| 画面 | URL | 概要 |
|------|-----|------|
| ホーム | / | 試験開始・面接開始・履歴表示 |
| 模擬試験 | /exam | 試験実施（語彙→読解→英作文） |
| 試験結果 | /results | 採点結果・解説・AI分析 |
| 面接練習 | /interview | 面接実施・採点結果 |

### 4.3 APIエンドポイント

| エンドポイント | メソッド | 概要 |
|--------------|--------|------|
| /api/generate-exam | POST | 模擬試験問題を生成 |
| /api/evaluate-writing | POST | 英作文を採点 |
| /api/analyze-results | POST | 試験結果を分析 |
| /api/generate-interview | POST | 面接問題を生成 |
| /api/evaluate-interview | POST | 面接回答を採点 |

## 5. 運用要件

### 5.1 API利用料金

- 使用モデル：Claude Opus 4.7
- 入力：$5 / 100万トークン
- 出力：$25 / 100万トークン

| 操作 | 1回あたりの費用目安 |
|------|-------------------|
| 模擬試験生成 | 約$0.08〜0.12 |
| 英作文採点＋弱点分析 | 約$0.05〜0.08 |
| 面接練習（生成＋採点） | 約$0.05〜0.08 |
| **1セット合計** | **約$0.15〜0.25** |

### 5.2 クレジット管理

- Anthropic Console（https://console.anthropic.com）の「Usage」で使用量を確認
- 「Billing」でクレジット残高・購入履歴を確認
- 自動チャージ設定を推奨（残高切れによるサービス停止を防止）

### 5.3 デプロイ

- GitHubのmainブランチへのpushで自動デプロイ
- 環境変数 `ANTHROPIC_API_KEY` をVercelのEnvironment Variablesに設定
